# Crime Analysis Dashboard

Overview :
An interactive dashboard built using Power BI to analyze crime patterns and trends, helping in making data-driven decisions.

Live Dashboard:
https://app.powerbi.com/groups/me/reports/4097e3da-31d9-40d9-afc4-0e025e43070a?ctid=b994aef6-35a1-414c-a3a8-d71709f2d8a5&pbi_source=linkShare

 Key Features:
- Crime distribution by type and location  
- Time-based trend analysis  
- KPIs for total crimes  
- Interactive filtering  

 Tools Used
- Power BI  
- DAX  
- Data Analysis  

Insights
This dashboard helps identify crime hotspots and patterns, enabling faster and smarter decision-making.
